BASIC WEB


Responsive website


In our first greater project we’ve been assigned to produce a responsive website with relevant content. We have been given art styles, in my case it was Bauhaus. By the end of this project I’ve learnt all parts of design process while building a website from mood board, style tile, wire framing, prototyping, testing and coding. Key to great results was to know basic design principles and applying them in practice.
