BASIC WEB


I’ve been introduced to the basic terminology within user interface design, digital communication, user testing, and responsive web design. I’ve learnt to code web pages with HTML and CSS as well as learnt basic image processing (Photoshop), vector drawing (Illustrator) and prototyping user interfaces and user experiences (XD).


BASIC UX


I’ve got basic understanding of the interaction between users and user interfaces, as well as the theories, tools and methods I can use for research, design and test developing a digital product. Data collection, prototyping, design sprint and much more…


BASIC ANIMATION


I’ve been introduced to concepts of basic animation, classic storytelling and character design as well as work with CSS animations and enhanced interactivity through JavaScript. Sprite animation, JavaScript controlled animation, onbording and much more…


BASIC CONTENT


I’ve learnt basics of concept development using video, dramaturgy, film language, media elements, recording and editing techniques. We focused on the design, programming and production of simple digital content solutions. And finally we had our first real campaign project for Foodsharing Copenhagen.
